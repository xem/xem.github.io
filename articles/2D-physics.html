Golfing a 2D physics engine in JS


November 2025


Introduction

I have a long love-hate story with physics in JS.

It all started in 2019, when I read this 134 pages book, summarized its contents in just 9 pages.

I used that to creatz a 1.57kb lib called Mini2DPhysics and the JS1K entry Emojysics.

The problem was that this book, as every book I could find at the time, didn't go beyond basic interactions between spheres and discs.

Even this 599 pages book released in 2022 by the same editor and some of the original authors, managed to be even more verbose in every aspect of 2D games, except one: advanced physics. And when I say "advanced physics", I'm not asking for really complex stuff, I just want to learn springs, fixed joints, hinge joints, and how to optimize speed and stability.

In 2021, 2022, 2023, 2024 and 2025 I tried many times to learn 3D physics because a lot of good books actually cover this topic, but I haven't managed to properly implement any of them in JS (so far)... and that was quite frustrating!

It's only in October 2025, after completing a new edition of JS13KGames, that I typed "2D physics JS" on Google and stumbled upon the holy Grail: this 2024 Reddit post announcing a complete Youtube playlist dedicated to learning 2D physics in JS. The youtube playlist made me realize that not only the code was already available and open-source on Gitlab, but a physical book actually released in August 2025!. And since I'm so enthusiastic (and probably one of the 10 person on Eart that could possibly be interested by this book), I got it. And I'm gonna summarize it and golf the source code here. Let's go!

My code will be published in public domain on the 2DPhysics Github repo, and a copy of the original Gitlab repo is also there for reference and for the live demos (thanks to Github Pages).

Chapter 1: Setting up a Canvas

Here we'll keep the HTML code to the absolute minimum: doctype, canvas and script tags. The canvas is called a and the 2D context x. 

<!doctype html>
<canvas id=a width=800 height=600 style="background:#eee"></canvas>
<script>
  c = a.getContext("2d");
</script>
