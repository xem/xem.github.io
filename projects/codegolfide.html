Codegolf IDE

February 2016 - October 2017

A Web IDE dedicated to HTML/JS code golfing, with live preview, Closure Compiler & RegPack integration, etc.

2017 edit: Closure Compiler and Regpack were broken, so I only kept the code editor and the preview, in fullscreen. Minification and RegPack are available separately in terser-online.
